URL Shortener Service

Welcome to url. This service uses GitHub Issues to store and redirect URLs.

How to create a short link:

  1. Create a new Issue in the repository.
  2. Title: Enter the full destination URL including http:// or https:// (e.g., https://google.com).
  3. Submit the issue and note the issue number.

Usage:

Use your link with the issue number:

https://ezharjan.github.io/url/{issue_number}

Example: if your issue number is 1, your link is:

https://ezharjan.github.io/url/1
